# Fit Image

**`.fit-image-contain`**

Put on wrapper of image so it contains it.

```html
<div class="fit-image-contain">
  <img src="shrek.jpg" />
</div>
```

**`.fit-image-cover`**

Put on wrapper of image so it covers it.

```html
<div class="fit-image-cover">
  <img src="shrek.jpg" />
</div>
```
