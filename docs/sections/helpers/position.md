## Offsetting

you can add a 0 to top,bottom,left,right horizontally or vertically with dem bois

### All sides

- **`.offset-auto`**
- **`.offset-0`**

### Horizontal or vertical sidez

- **`.offset-x-auto`**
- **`.offset-x-0`**
- **`.offset-y-auto`**
- **`.offset-y-0`**

### Indivisual sidez

- **`.offset-top-auto`**
- **`.offset-top-0`**
- **`.offset-left-auto`**
- **`.offset-left-0`**
- **`.offset-bottom-auto`**
- **`.offset-bottom-0`**
- **`.offset-right-auto`**
- **`.offset-left-0`**

[More info on the css property](https://www.w3schools.com/css/css_positioning.asp)
